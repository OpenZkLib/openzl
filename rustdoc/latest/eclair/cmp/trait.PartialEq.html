Partial Equivalence Relations

Required Methods§

source

fn eq(&self, rhs: &Rhs, compiler: &mut COM) -> Bool<COM>

Returns true if self and rhs are equal.

Provided Methods§

source

fn ne(&self, other: &Rhs, compiler: &mut COM) -> Bool<COM>where
    Bool<COM>: Not<COM, Output = Bool<COM>>,

Returns true if self and rhs are not equal.

source

fn assert_equal(&self, rhs: &Rhs, compiler: &mut COM)where
    COM: Assert,

Asserts that self and rhs are equal.

Implementation Note

This method is an optimization path for the case when comparing for equality and then asserting is more expensive than a custom assertion.
