) -> Option<&<<C as HashConfiguration<()>>::LeafHash as LeafHash<()>>::Output>

Returns the leaf digest at the given index.

Implementation Note

This method is allowed to return None even if index is less than the current length of the tree. See position for more.

source

fn position(
    &self,
    leaf_digest: &<<C as HashConfiguration<()>>::LeafHash as LeafHash<()>>::Output
) -> Option<usize>

Returns the index of the leaf_digest if it is contained in self.

Implementation Note

This method is allowed to return None even if leaf_digest was inserted with a call to push_digest. This method need only return an index for leaves which are inserted with a call to push_provable.

fn maybe_push_provable_digest<F>(
    &mut self,
    parameters: &Parameters<C, ()>,
    leaf_digest: F
) -> Option<bool>where
    F: FnOnce() -> Option<<<C as HashConfiguration<()>>::LeafHash as LeafHash<()>>::Output>,

Checks if a leaf can be inserted into the tree and if it can, it runs leaf_digest to extract a leaf digest to insert, returning None if there was no leaf digest and returning Some(false) if the tree is at capacity. If this method is successful, the digest inserted will have an accompanying proof that can be returned by a call to the path method.

fn remove_path(&mut self, index: usize) -> bool

Removes a single path at the given index, returning true if it was removed.

Implementation Note

This method may return false for arbitrary inputs and is only an optimization path for removing unused memory.
