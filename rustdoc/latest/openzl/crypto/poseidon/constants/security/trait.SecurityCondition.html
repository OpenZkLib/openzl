Expand description

Security Conditions

This trait is defined for each known attack on Poseidon. Details of these attacks is present in section 5.5 of the Poseidon paper.

    width: f32,
    partial_rounds: f32,
    modulus_bits: f32,
    security_level: f32
) -> usize

Computes the lower bound on the secure number of full rounds required for a poseidon with width and partial_rounds constants and prime field of size modulus_bits for security_level bits of security.
