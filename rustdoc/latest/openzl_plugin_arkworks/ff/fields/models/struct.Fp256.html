Struct openzl_plugin_arkworks::ff::fields::models::Fp256

pub struct Fp256<P>(pub BigInteger256, _);
Available on crate feature ff only.
Expand description

Represents an element of the prime field F_p, where p == P::MODULUS. This type can represent elements in any field of size at most 256 bits.
