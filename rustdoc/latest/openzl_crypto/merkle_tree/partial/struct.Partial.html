pub fn leaf_digests(&self) -> &[LeafDigest<C>]

Returns the leaf digests currently stored in the merkle tree.

Note

Since this tree does not start its leaf nodes from the first possible index, indexing into this slice will not be the same as indexing into a slice from a full tree. For all other indexing, use the full indexing scheme.
