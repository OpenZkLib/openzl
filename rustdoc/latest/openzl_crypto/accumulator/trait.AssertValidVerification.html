Accumulator Membership Model Validity Assertion

For situations where we just want to assert validity of the membership proof, we can use this trait as an optimization path for Model::verify. See assert_valid for more details.

Optimization

In compilers where assertions for more complex statements other than booleans being true, this function can provide an optimization path to reduce the cost of assertion.
