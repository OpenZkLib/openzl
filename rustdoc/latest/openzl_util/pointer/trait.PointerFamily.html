fn claim(strong: Self::Strong) -> T

Claims ownership of the underlying owned value from strong.

Panics

This method can only panic if there are other outstanding strong pointers. This method will still succeed if there are other outstanding weak pointers, but they will all be disassociated to strong.

source

fn downgrade(strong: &Self::Strong) -> Self::Weak

Returns a new weak pointer to strong.

source

fn upgrade(weak: &Self::Weak) -> Option<Self::Strong>

Tries to upgrade weak to a strong pointer, returning None if there is no strong pointer associated to weak.

source

Checks if two strong pointers point to the same allocation.
